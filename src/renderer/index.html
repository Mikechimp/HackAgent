3

Load the extension

Click "Load Temporary Add-on..." and select the downloaded hackagent.xpi file

Use Firefox Developer Edition

Download Firefox Developer Edition or Firefox Nightly which allow unsigned extensions

mozilla.org/firefox/developer
2

Disable signature enforcement

In the address bar, go to:

about:config

Search for xpinstall.signatures.required and set it to false

3

Install the extension file

Go to about:addons (or press Ctrl+Shift+A)

Click the gear icon ⚙ → "Install Add-on From File..."

Select the downloaded hackagent.xpi
